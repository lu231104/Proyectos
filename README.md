# Dashboard de Accidentes de Tránsito - SUTRAN

Proyecto de análisis y visualización de accidentes de tránsito en carreteras del Perú utilizando **SQL Server** y **Power BI**.

El dashboard permite identificar patrones de accidentes, departamentos con mayor incidencia, horarios críticos y métricas de mortalidad y lesividad.

---

## Tecnologías

* SQL Server
* Power BI
* DAX
* Power Query

---

## Dataset

Los datos fueron obtenidos desde un repositorio público de accidentes reportados por SUTRAN.

## Proceso

* Limpieza y transformación de datos
* Creación de vistas SQL
* Exportación y modelado en Power BI
* Creación de medidas DAX y tablas auxiliares
* Desarrollo de dashboards interactivos

---

## Dashboards

### Resumen General

* Total de accidentes, heridos y fallecidos
* Modalidad más frecuente
* Departamento más peligroso
* KPIs de mortalidad y lesividad

### Análisis Regional

* Top departamentos con más accidentes
* Mapa interactivo
* Tramos y kilómetros más peligrosos

### Análisis Temporal

* Accidentes por hora
* Variación mensual
* Comparativa anual
